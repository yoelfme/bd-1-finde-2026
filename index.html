Paso a paso

Cómo fluye el trabajo con Git

Piensa en Git como un álbum de fotos de tu proyecto: primero miras qué cambió, luego eliges qué guardar y al final dejas una foto con un mensaje claro.

  1. git init

    Prepara la carpeta para que Git pueda recordar el historial. Solo se hace una vez al empezar.

  2. git status

    Te muestra el estado actual: qué archivos son nuevos, cuáles editaste y cuáles ya están listos para guardar.

  3. git diff

    Enseña el detalle de los cambios: líneas que agregaste o borraste, antes de guardarlas.

  4. git add

    Elige qué cambios van en la próxima “foto”. Puedes agregar un archivo o varios.

  5. git commit

    Guarda esa foto en el historial con un mensaje corto que diga qué hiciste y por qué.

  6. git show

    Abre una foto ya guardada para revisar qué trajo ese commit.

Truco rápido: usa status y diff antes de cada commit. Así evitas guardar algo que no querías.

Git + GitHub

Cómo trabaja un equipo a distancia

Git vive en tu computadora. GitHub es el lugar en la nube donde el equipo deja la versión compartida del proyecto.

  1. git push

    Sube tus commits a GitHub para que el resto del equipo los vea.

  2. git fetch

    Pregunta a GitHub si hay novedades y las descarga, pero no las mezcla todavía con tu trabajo.

  3. git pull

    Trae los cambios de GitHub y los aplica a tu copia. Es “fetch + integrar” en un solo paso.

Rutina de equipo: empieza el día con pull, trabaja, guarda con commit y termina con push. Así todos ven la misma versión.

Mapa rápido de comandos

Una hoja de atajos cuando no recuerdes qué comando usar.

  • init

    Arrancar el historial del proyecto.

  • status

    Ver el panorama: qué está pendiente.

  • diff

    Mirar el detalle de los cambios.

  • add

    Elegir qué va en el próximo guardado.

  • commit

    Guardar una versión con mensaje.

  • show

    Revisar un commit ya guardado.

  • push

    Subir tu trabajo a GitHub.

  • fetch

    Enterarte de novedades sin aplicarlas aún.

  • pull

    Traer e integrar lo nuevo del equipo.
